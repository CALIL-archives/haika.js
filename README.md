## Development

python -m SimpleHTTPServer 9998
cd haika/
python manage.py runserver 9999
grunt server
grunt esteWatch

### Dependencies
- `node.js` >= 11.0
- `npm install -g gulp`

### Deploy

```
$ git clone git@github.com:CALIL/haika.js.git
$ cd haika
$ bower install
$ npm install
$ gulp
```

### Commands

- `gulp` concats all files for test.
