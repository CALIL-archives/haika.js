## Development

### Dependencies
- `node.js` >= 11.0
- `npm install -g grunt-cli`

### Deploy

```
$ git clone git@github.com:CALIL/haika.js.git
$ cd haika
$ npm install
$ grunt
```

### Commands

- `grunt` concats all files for test.
- `grunt watch` executes `grunt` each time at updating JavaScript files.


### Testing

Open `test/index.html`.
