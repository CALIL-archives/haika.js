## Development


### Dependencies
- `node.js` >= 11.0
- `npm install -g gulp`

### Deploy

```
$ git clone git@github.com:CALIL/haika.js.git
$ cd haika
$ bower install
$ npm install
$ gulp
```

### Commands

- `gulp`
